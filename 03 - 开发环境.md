## python

1.安装tk

`sudo pacman -S tk`

2.pip镜像

`pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple`

3.虚拟环境

```bash
python -m venv .venvs/learn
source /home/0x2a/.venvs/learn/bin/activate # 将此行写入 .zshrc
```

4.jupyterlab

```bash
pip install jupyterlab jupyterlab-widgets
jupyter lab --generate-config
pip install jupyterlab-language-pack-zh-CN
```

5.注册内核

```bash
python -m ipykernel install --user --name learn --display-name LearnAI

--name learn # 虚拟环境名称
--display-name LearnAI # 内核名称
```
